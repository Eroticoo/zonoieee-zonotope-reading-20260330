# Zonotope Paper Deep Reading

This repository stores five screenshot-backed Markdown reading reports for constrained-zonotope and zonotope-related papers.

Chronology used in the reports:

- Main nonlinear CZ estimation line: `2020 Automatica -> 2021 Automatica -> 2022 Automatica -> 2025-03-31 arXiv`
- Parallel branch: `2025-05-23 Automatica (Line zonotopes)` extends the CZ idea toward unbounded/descriptor systems and AFD, rather than continuing the nonlinear discrete-time estimation line directly.

Included outputs per paper folder:

- `report.md`
- `report.rendered.md`
- `wechat_assets_manifest.txt`
- `images/`

Local source PDFs and extracted `source_text.txt` files are intentionally not tracked in this public repository.
